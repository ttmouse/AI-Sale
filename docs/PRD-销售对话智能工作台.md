# PRD：销售对话智能工作台 — 场景演示页面

> 版本: v1.0 | 日期: 2026-07-22 | 状态: 定稿

---

## 1. 产品定位

基于文档《☆销售AI助手-功能清单-展示页面》的设计理念，在现有 AI-Sale Demo 系统中新增一个**销售对话智能工作台场景演示页**。

核心表达：**销售 AI 助手不是在聊天框中替销售回复客户，而是在每一次客户对话中，自动沉淀客户画像、推进销售阶段、推荐最佳动作，并把关键承诺转化为可执行流程。**

---

## 2. 页面结构：三栏布局

```
┌─────────────────────────────────────────────────────────────────┐
│ 顶部导航：场景选择器（阶段tab + 场景下拉） | 阶段进度条          │
├──────────────┬──────────────────────────┬────────────────────────┤
│ 左侧说明区   │ 中间手机对话区            │ 右侧AI智能注解区       │
│              │                          │                        │
│ 当前阶段     │ 微信风格聊天界面           │ 客户画像更新卡         │
│ 场景名称     │ 客户消息气泡              │ 项目推进状态卡         │
│ 销售痛点     │ 销售回复气泡              │ 流程动作识别卡         │
│ AI功能亮点   │ AI推荐回复卡片            │ 知识卡推荐区           │
│ 演示目标     │ 快捷操作按钮              │ 下一步建议卡           │
└──────────────┴──────────────────────────┴────────────────────────┘
```

### 2.1 左侧：场景说明区

- **当前销售阶段**：P1~P7，含阶段名称和进度
- **场景名称**：当前演示的子场景名称
- **销售痛点**：该场景解决什么问题
- **AI 功能亮点**：AI 在该场景中做的具体事情（列表）
- **演示目标**：该场景演示完客户应感知到的核心价值

### 2.2 中间：手机对话区

模拟企微/微信风格的手机聊天界面：

- 客户头像 + 名称
- 客户消息气泡（灰色，左对齐）
- 销售回复气泡（蓝色，右对齐）
- AI 推荐回复卡片（可选 A/B/C 选项）
- 系统提示气泡（居中，小字号）
- 时间分隔线
- 对话自动播放/手动逐步播放控制

### 2.3 右侧：AI 智能注解区

5 类实时更新的卡片：

| 卡片类型 | 内容 |
|----------|------|
| **客户画像更新** | 客户来源、类型、分类、关注点、画像完整度 |
| **项目推进状态** | 当前阶段、已完成/待完成事项、阶段变化 |
| **流程动作识别** | 可创建的动作按钮（日程/任务/流程） |
| **知识卡推荐** | 根据上下文推荐的产品卡/案例卡/ROI卡等 |
| **下一步建议** | 建议的动作 + 推荐话术 |

## 4. 交互流程

### 4.1 场景切换
1. 顶部导航栏显示 7 个阶段 Tab
2. 点击 Tab → 显示该阶段下的子场景列表
3. 点击具体场景 → 重置中间对话和右侧注解区
4. 支持「上一场景」/「下一场景」快速切换

### 4.2 对话播放
1. 场景加载后，自动从第一条消息开始播放
2. 每条消息按间隔逐条出现（销售回复 → AI 推荐 → 客户回复...）
3. 支持暂停/继续、快速跳过
4. 消息播放时，右侧注解区同步更新

### 4.3 注解区更新
- 对话每推进一轮，右侧对应的卡片数据同步刷新
- 客户画像完整度以动画进度条展示（0% → 30% → 70%...）
- 阶段变化以醒目标签提示（P2 → P3）
- 可创建动作按钮可点击（模拟 toast 反馈）

## 5. 非功能性需求

| 项目 | 要求 |
|------|------|
| **技术栈** | 纯前端 HTML/CSS/JS，无框架依赖 |
| **兼容性** | 现代浏览器（Chrome/Firefox/Safari/Edge） |
| **数据** | 所有场景数据定义为 JS 模块，无后端依赖 |
| **测试** | 引擎层单元测试覆盖率 ≥ 90% |
| **集成** | 在主页场景网格中注册，可通过主页跳转 |
| **响应式** | 桌面端优先（三栏），1280px 宽屏优化 |
