# AI-Sale · 项目档案

## 一句话定位
为**高客单、非标、长周期、强线下服务依赖**的销售场景做 AI 赋能。首个落地行业：智能厨具（智能炒菜机 / 商用厨具 / 非标餐饮设备）。

## 项目目标
- **内部使用**：把跑通的销售流程系统化、AI 化，让新人快速上手、资深销售经验可沉淀复用
- **行业复用**：方案可向同类高客单非标行业（教育招生、医美、企业服务、定制装修等）横向复制商业化

## 核心命题
**AI 不替代销售，专注全链路提效、标准化、可复制。** 半自动辅助、不替代人工。

## 行业核心属性（落地基础）
- 高客单、非标定制、强依赖线下服务（试菜 / 上门 / 配套）
- 销售周期长（1–2 个月+），客户决策慢、易断联
- 销售能力分层严重：新人专业弱、跟进随意；资深经验无法沉淀
- 客户类型复杂：餐饮门店 / 食堂 / 连锁机构 / 创业者，需求差异化大
- 企业诉求：标准化销售流程、降低培训成本、盘活沉睡线索、提升转化

## 8 大 AI 落地场景（当前方案版本 v1）

| # | 场景 | 核心痛点 | AI 能力 | 价值归类 |
|---|------|---------|---------|---------|
| 1 | 销售随身智能知识库 | 产品参数复杂查资料慢 | 实时解答参数 / 配置 / 报价 / 适配方案 | 销售赋能 |
| 2 | 企微实时话术参谋 | 临场不会接话只会发资料 | 侧边栏实时推荐话术 + 跟进动作（半自动） | 销售赋能 |
| 3 | 客户分层 + 智能 SOP 跟进 | 长周期客户遗忘、流失 | 自动标签分层、阶段定时提醒触达 | 客户运营 |
| 4 | 聊天记录 AI 复盘沉淀 | 优秀经验无法复用 | 提炼高转化话术 / 流失原因，形成 SOP | 资产沉淀 |
| 5 | 客户个性化数据应答 | 发货 / 售后 / 试菜排期问不准 | 对接内部系统，实时查客户专属信息 | 客户运营 |
| 6 | AI 新人模拟培训对练 | 培训成本高、新人不敢上 | AI 模拟客户实战对练 + 点评 | 销售赋能 |
| 7 | 线索智能初筛评级 | 线索量大筛选效率低 | 自动判定意向 / 预算 / 类型，优先推送高价值 | 客户运营 |
| 8 | 行业方案智能生成 | 销售不会搭配方案 | 按客户类型自动生成配套方案 / 案例 / 报价 | 资产沉淀 |

## 三大价值归类（汇报口径）
- **销售赋能类**：知识库答疑、实时话术参谋、AI 模拟培训
- **客户运营类**：线索筛分、SOP 自动跟进、个性化数据应答
- **资产沉淀类**：聊天复盘、话术沉淀、方案智能生成

## 当前交付物
- `index.html` — 智能厨具行业 AI 销售赋能全场景解决方案展示页（Tailwind + animate.css，单页静态站，可作汇报 / 客户演示用）

## 技术与协作偏好（继承豆爸全局）
- 前端：React + TypeScript + Vite + Tailwind；展示型页面可用纯 HTML + Tailwind CDN（如本项目首页）
- 数据：Supabase 优先，SQLite 兜底
- 自动化：N8N + MCP
- 协作：简洁直接、有主见、先做再说、数据说话、沉淀复利
